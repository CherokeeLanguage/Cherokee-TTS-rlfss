Tacotron with Location Relative Attention

A PyTorch implementation of Location-Relative Attention Mechanisms For Robust Long-Form Speech Synthesis

This page contains audio samples from https://github.com/bshall/Tacotron: an implementation of
"Location-Relative Attention Mechanisms For Robust Long-Form Speech Synthesis", E. Battenberg, (2020).

Notable differences from the paper:

  • Trained using a batch size of 64 on a single GPU (using automatic mixed precision)
  • Used a gradient clipping threshold of 0.05 as it seems to stabilize the alignment with the smaller batch size.
  • Used a different learning rate schedule (again to deal with smaller batch size).
  • Used https://github.com/bshall/UniversalVocoding to generate waveforms instead of WaveRNN. The vocoder was trained on the ground-truth spectrograms.
  • Used 80-bin (instead of 128 bin) log-Mel spectrograms.
